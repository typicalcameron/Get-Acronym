# Get-Acronym
 ## Description:
 * Exercism Exercise - https://exercism.org/tracks/powershell/exercises/acronym
 * Converts a phrase to its acronym.

 # Example:
| Input	| Output |
| -------- | ------- |
| As Soon As Possible | ASAP |
| Liquid-crystal display | LCD |
| Thank George It's Friday! | TGIF |
